# Update Notes & Media

There are several methods to adding notes to trades as well including media. Media uploads are limited to PRO members only.



### Add notes from Active Positions interface

Using the Active Positions interface, trader's can add standalone notes that are displayed in the section showing the trader's trade events (open, average, trim, close, and notes). Media can be also be included with notes and are limited to [PRO members](https://nvstly.com/settings/membershipl) only.

<figure><img src="../../.gitbook/assets/image (260).png" alt=""><figcaption><p>Select <code>Update</code> from the dropdown menu under <code>Status</code> to add standalone notes</p></figcaption></figure>

### Add notes from Trade Insight

The author of a trade can use the [Trade Insight](../../info/readme/user-interfaces-overview.md#trade-insight) interface to add standalone notes that are displayed in the section showing the trader's trade events (open, average, trim, close, and notes). Media can be also be included with notes and are limited to [PRO members](https://nvstly.com/settings/membership) only.

<figure><img src="../../.gitbook/assets/image (257).png" alt=""><figcaption><p>Select <code>Update</code> from the dropdown menu under <code>Status</code> to add standalone notes</p></figcaption></figure>
